Button Row

Author Erik van Velzen, erik@polderknowledge.nl, erik@evanv.nl
Date 21-05-2017
Repository name button-row

Goals

When displaying buttons:

  • Buttons should space correctly amongst eachother
  • Buttons should fit exactly in their common container: the buttons should touch the padding of the container on all sides
  • It should space correctly when the buttons don't fit on one row

Implementation

Resize your window to view the behaviour.

Base

Many More Buttons And one with a very long label 
        <style>
            .button-row {
                display: flex; /* flexbox resolves many spacing issues for us */
                flex-wrap: wrap;
                align-items: flex-start;
                margin: -.2em; /* negative margin is equal to the positive margin of the buttons */
            }
            .button-row > * {
                margin: .2em;
            }
        </style>

        <div class="container">
            <div class="button-row">
                <button class="button">Very</button>
                <a class="button">Many</a>
                <a class="button">More</a>
                <a class="button">Buttons</a>
                <a class="button">And one with a very long label</a>
            </div>
        </div>

Accessibility

This pattern has no specific accessibility demands. For the buttons themselves, the usual guidelines apply:

  • Stick to the standard elements A and BUTTON (for all users with a disability)
  • Don't use icon-only buttons (for screen readers)
  • Make sure your theme uses a distinct :focus style (for users unable to use point-and-click)
